CEGE0042 Coursework - TfL Bike Sharing Analysis

This project analyses and forecasts hourly bike-sharing demand across London's Santander Cycles network using spatio-temporal methods. It compares a seasonal time series model (SARIMA) with a space-time autoregressive model (ST-ARIMA) using TfL open data from January to March 2025, producing 15 visualisations and a model performance comparison.

Data:
- 6 CSV files from cycling.data.tfl.gov.uk/usage-stats (weekly files from January to March 2025)
